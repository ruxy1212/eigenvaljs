About

This is a Javascript web application that assists in solving for eigenvalues by providing the quadratic/cubic equation solution. I decided to write my own application when I needed to help someone solve for eigenvalues (complex), but I was without a scientific calculator; I went online but only found a solution for quadratic only (at least, from Google search results), so the thought came to my mind. Now, the next time, I can do it online even with a torchlight phone that has a browser which supports Javascript. If you like my work, please give me a Star
